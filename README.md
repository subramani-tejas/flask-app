https://linuxize.com/post/how-to-install-flask-on-ubuntu-20-04/

# VS Code (terminal):

python3 -V (returns Python 3.8.10)
sudo apt install python3-venv
mkdir flask_app && cd flask_app
python3 -m venv venv
source venv/bin/activate
pip install Flask
python -m flask --version
